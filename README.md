# Portfolio — Aryan Kandula

**Live site:** [aryan-kandula.github.io/portfolio](https://aryan-kandula.github.io/portfolio/)

---

A personal portfolio website I built from scratch using HTML, CSS, and vanilla JavaScript. Designed to showcase my projects, skills, and background as a Computer Science student and developer, featuring an Apple-inspired, glassmorphic dark theme.

## What's in it

- **Animated hero section** with an iOS-inspired dark theme and an interactive HTML5 canvas background featuring floating particles and orbs.
- **Skills section** with an infinite-scroll ticker and animated progress bars that trigger on scroll.
- **Projects section** covering my main work — full-stack Django web apps, Java desktop software, vanilla JS tools, and SQL analytics.
- **AI & Modern Development section** detailing my AI-assisted workflows, prompt engineering approach, and formal AI studies.
- **Education section** showing my current A.S. at FTCC, planned transfer to UNCP, and a core strengths cloud.
- **Glassmorphic navigation bar** that highlights the active section as you scroll and smoothly jumps to each section on click.
- **Horizontal marquee sections** highlighting my key skills and roles.
- **Smooth reveal animations** on scroll using `IntersectionObserver`.
- **Fully responsive** layout for mobile and desktop.

## Built with

- HTML5
- CSS3 — custom properties, glassmorphism, keyframe animations, CSS grid/flexbox
- Vanilla JavaScript — `IntersectionObserver`, canvas animations, scroll events, ripple effects
- Google Fonts — Syne, DM Mono, and Fraunces

## Run locally

Just open `index.html` in any browser. No build step, no dependencies.
